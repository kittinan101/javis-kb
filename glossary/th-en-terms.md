# 📖 Glossary: ศัพท์ไทย ↔ อังกฤษ

> ไฟล์นี้สำคัญต่อการค้นหาภาษาไทยของ Javis — เมื่อ user ถามด้วยคำไทย ระบบใช้ตารางนี้ map เป็นคำอังกฤษเพื่อค้นเอกสาร (และกลับกัน)
>
> **กติกา:** เจอศัพท์ใหม่ในเอกสาร → เพิ่มที่นี่ทันที / เรียงตามหมวด / คำไทยหลายแบบใส่คั่นด้วย `/`

## Authentication & Users

| ไทย | อังกฤษ |
|---|---|
| เข้าสู่ระบบ / ล็อกอิน | login, sign in |
| ออกจากระบบ | logout, sign out |
| สมัครสมาชิก / ลงทะเบียน | register, sign up |
| ยืนยันตัวตน | authentication |
| กำหนดสิทธิ์ / ตรวจสอบสิทธิ์ | authorization |
| รหัสผ่าน | password |
| ลืมรหัสผ่าน | forgot password, password reset |
| ผู้ใช้ / ผู้ใช้งาน | user |
| บทบาท | role |
| สิทธิ์ / สิทธิ์การเข้าถึง | permission, access right |
| โปรไฟล์ | profile |
