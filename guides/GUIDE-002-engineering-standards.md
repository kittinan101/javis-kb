> Import จาก Notion Wiki (2026-07-05) — หลาย section ยังเป็นโครงรอทีมเติม (ทำเครื่องหมาย ⚠️ ไว้)

มาตรฐานการเขียน code และ workflow ของทีม — ทุกคนต้องทำตาม

## Git Workflow

- Branch naming: `feature/HEY-xxx-description`, `fix/BUG-xxx-description`
- Commit message: อ้างอิง Task ID เสมอ เช่น `feat: add checkout API (HEY-12)`
- PR ต้องมี reviewer อย่างน้อย 1 คนก่อน merge
